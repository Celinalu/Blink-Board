# Blink Board
A flashing carrot blink board!!!
This carrot will not help your eyesight but will help you see in the dark with it's led lights. This is a pocket sized, carrot shaped PBC that uses a oscillator circuit to make LEDs flash. Since I only used transistors and capacitors there is no code needed for this project.

# How to use it 
All you need to do is to connect the blink board to a battery, and then flip the switch to start. If the lights are flicking then that means its working!

I made this since it was fairly strightforward project as someone creating hardware for the first time and I thought it was a cute and interesting accessory to create.
